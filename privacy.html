1. What We Collect

BizTrack may store order information, product information, finance records, language preferences, and cookie consent choices entered by users.

2. How We Use Data

The data is used only to provide order management, product tracking, finance tracking, and user preference features for this coursework project.

3. Cookies and Local Storage

This website uses browser localStorage to remember cookie consent choices and user preferences. Users can clear this data through their browser settings.

4. Third-Party Services

This project may be hosted on Vercel. Hosting providers may process basic technical data such as IP address, browser type, and access logs.

5. User Choices

Users can accept, reject, or allow only necessary storage through the cookie banner. The selected choice is stored locally in the browser.

6. Contact

For privacy-related questions, please contact the CPT304 coursework project team.
